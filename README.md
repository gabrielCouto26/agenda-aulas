# Agenda Aulas

## MVP do projeto de semestre da faculdade

### Instalação

* Instale o `Ruby v2.7.5`
  
* Instale o `mysql2`

* Execute o comando `bundle install`


* Execute o comando `rails db:create`
  
* Execute o comando `rails db:migration`

<br />

### Rodando o projeto

* Execute o comando `rails s`
